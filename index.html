Introduction to Vim

Pros


  • Quickly navigate and edit a text document using only the keyboard (no mouse)
  • Extremely Customizable - Powerful and vast library of plugins
  • Available almost anywhere - Edit local and remote (SSH) files and on systems without a GUI editor, e.g. Raspberry Pi, arduino
  • FREE - Never pay for another third-party text editor
  • Allows for editing files that GUI editors can't edit due to file permissions - "sudo vim"
  • Modular - key commands can be combined in different ways to do powerful stuff (more on this later)

Cons


  • Steep learning curve for users used to GUI editors like Coda or Sublime Text
  • No bells and whistles like syntax highlighting or indentation out of the box. Have to search for plugins to get extra features.
  • Plugins may sometimes conflict with one another and you'll have to figure out which one(s) are the culprit

3 Vim modes


Command mode - navigating and manipulating text


Insert mode - writing text as if in a normal text editor


Visual mode - selecting text using movement keys

How to switch between modes


  • Command mode - this is the default mode. To switch back to it from another mode, press the 'Esc' key
  • Insert mode - 'i' key
  • Visual mode - 'v', 'Ctrl-v', or 'Shift-v'. (We'll go into the differences later)

Learning the Command Mode

Single character movement


h, j, k, l

left, down, up, right

Multi-character movement


  • w - move forward one word
  • b - move back one word
  • e - move to end of word
  • 0 - move to beginning of line
  • ^ - move to the first non-whitespace character of line
  • $ - move to end of line
  • f-[key] - move forward to next instance of [key]
  • F-[key] - move backward to last instance of [key]

Multi-line movement


  • gg - move to the first line of the document
  • G - move to the last line of the document
  • [number]-G - move to the line number in the document. Example: 42G moves your cursor to line 42. (Enable line numbers in Vim to make this easier to use.)
  • Ctrl-e/y - scroll one line down/up
  • Ctrl-u/d - scroll up/down
  • Ctrl-f/b - page forward/back
  • H/M/L - move cursor to top/middle/bottom of screen

Combining numbers with movement


  • 5w - move forward 5 words
  • 3j - move down 3 lines
  • d4b - delete backwards 4 words

Manipulating text


  • x - delete character under cursor
  • r-[key] - replace character under cursor with [key]
  • ~ - toggle lowercase/uppercase of letter under cursor
  • u - undo last change
  • Ctrl-r - redo last change
  • . - Repeats the last command you did

Copy and paste


  • y-[movement] - copy (or yank as it's referred to in Vim)
  • d-[movement] - delete (while yanking)
  • yy - copies line that cursor is on
  • dd - delete line that cursor is on
  • dw - delete word under cursor
  • d$ - delete to end of line
  • di( - delete inner parentheses. Delete everything inside the parentheses
  • y0 - yank to beginning of line
  • p - paste after cursor the last yanked item
  • P - paste before cursor the last yanked item

Search (and move to)


  • /keyword[Enter] - search forward
  • ?keyword[Enter] - search backwards
  • n - next match
  • N - previous match

Search (and replace)


  • :%s/word1/word2/g - replace every instance of 'word1' with 'word2' for entire document
  • :%s/word1/word2/gc - replace every instance of 'word1' with 'word2' for entire document but ask for confirmation for each one
  • :5,12s/word1/word2/g - replace 'word1' with 'word2' for only lines 5 through 12
  • :5,12s/word1\//word2/g - replace 'word1/' with 'word2' using \ to escape a special character for only lines 5 through 12

Other commands


  • :w - Save the file
  • :q - Quit the file
  • :wq - Save and quit
  • :![terminal command] - Run a terminal command while in Vim. Example: :!ls

Learning the Insert Mode

Ways to enter Insert mode


  • i - most common way. Start typing at the cursor
  • I - Move to beginning of the line and enter Insert mode
  • a - Start typing one character after the cursor
  • A - Move to end of the line and enter Insert mode
  • s - replace character under cursor and then enter Insert mode
  • o/O - open new line below/above cursor
  • cw - replace word under cursor and then enter Insert mode
  • ci" - change inner quotes. Delete everything inside the quotes and enter Insert mode
  • C - Delete rest of line from cursor and enter Insert mode
  • R - enter Replace mode

Faster editing in Insert mode


  • ctrl-w - delete backwards one word
  • ctrl-u - delete everything on the line before the cursor
  • ctrl-a - move to beginning of line
  • ctrl-e - move to end of line

Learning the Visual mode

3 types of Visual mode


  • v - select text starting from the initial position of cursor to ending position
  • Ctrl-v - select blocks of text by column
  • Shift-v - select lines of text

Commands while in Visual mode


  • Shift-< - indent the line(s) backward
  • Shift-> - indent the line(s) forward
  • J - combine all lines
  • d - delete all
  • y - yank all

Vim Buffers

  • :e [filename] - open another file in a buffer
  • :ls - list all files in the buffer
  • :b-[number] - switch to the file associated with the buffer number

Useful Vim plugins

  • Easymotion - Move around Vim quickly
  • IndentHTML - Automatic HTML Indentation
  • JShint - JavaScript linter
  • MatchTagAlways - HTML tag matching highlighting
  • Minibufexpl - See Buffers as tabs
  • NERDtree - File explorer
  • NeoBundle - Plugins manager for Vim
  • Seek - Better alternative to Vim's built-in 'f' move to character
  • TComment - Quickly comment/uncomment code
  • Unite - Fuzzy file searching and code searching
  • vim-colorschemes - lots of color schemes to choose from
